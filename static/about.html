We Rescue Songs

  • This is the bed music for The Witching Hour Tone Club
  • We bought a bunch of old 78s and a player for them at Goodwill
  • We digitize all the records
  • While recording, we test to see if they are in Shazam — we call this "The Shazam Test"
  • If they are NOT in Shazam, they pass
  • We now assume this song has never been digitized
  • Seeing as how these records are mostly 100+ years old... we now commit to rescuing it
  • We wash & dry the record, take a pic of the label, record it, type in the metadata — or all we can get, sometimes the labels are in bad shape
  • We keep the original recording as-is, make a copy, then clean up that copy
  • So we have the original — if you hate our mix, cool, we can send you the file and you can do it yourself
